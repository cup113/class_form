# 灵动课表

## 使用方法

1. 如果是第一次使用，编辑 `config.json`，主要更改`"日程表"`、`"课程时间"`字段。
2. 点击`class_form.exe`（或者配置 Python 环境后运行 `pythonw class_form.py`，即可运行。
3. 结束任务：直接关闭，或者借助任务管理器。
